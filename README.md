# Web Calculator

A simple and interactive web-based calculator built using HTML, CSS, and JavaScript.

## Features

* Perform standard arithmetic operations (addition, subtraction, multiplication, division)
* Live history tracking display above the input screen
* Percentage conversion helper button (%)
* Smart decimal point validation preventing multiple dots in a single operand
* Fully responsive layout with interactive hover effects and visual transitions
* Sign toggling functionality (+/-) to switch numbers between positive and negative

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Use

1. Click on the numeric keys to type out your first number
2. Choose an arithmetic operator button (+, -, *, or /)
3. Input your second number using the keypad
4. Click the green equals (=) button to instantly calculate the result
